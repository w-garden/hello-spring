# login

### 2022-12-22 추가

#### 로그인 요구사항
1. 홈 화면 -로그인 전
   - 회원가입
   - 로그인
2. 홈 화면 - 로그인 후
   - 본인이름(~님 환영합니다)
   - 상품 관리
   * 로그 아웃
3. 보안 요구사항
   - 로그인 사용자만 상품 접근, 관리 가능
   - 로그인 하지 않는 사용자가 접근하면 로그인 화면으로 이동
4. 회원가입, 상품관리
